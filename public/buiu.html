Protocolo Descentralizado ONPOSTT

Nosso protocolo é uma solução inovadora semelhante à rede Bitcoin, mas utiliza uma abordagem baseada em Transmissão por Relays e uma blockchain leve voltada para redes sociais e outras aplicações. Os dados são armazenados de forma descentralizada em um banco de dados PostgreSQL, assinados e criptografados com a chave privada do usuário.

Características Principais

  • Execução em Node.js
  • Uso de banco de dados PostgreSQL em vez de arquivos JSON
  • Blocos armazenados como registros no banco, com ID e chave pública (pubkey)
  • Consultas organizadas com SQL, filtros e parâmetros personalizados
  • Altamente escalável
  • Baseado em WebSocket e comunicação P2P

Segurança e Privacidade

Com criptografia baseada em chaves privadas, os usuários têm total controle sobre seus dados. A descentralização garante resistência à censura e elimina pontos únicos de falha.

Escalabilidade e Flexibilidade

A estrutura modular com PostgreSQL permite fácil integração com outras tecnologias e crescimento escalável sem comprometer a eficiência.

        onpostt.connect(['ws://localhost:3000', 'ws://localhost:4000']);

        async function post() {
            var privateKey = '21e28dfffa49daf6373527c579ee16dea1afd7c8a2f95d9eb2e6aeb0a8d6d3d2';
            var pubkey = onpostt.generatePublicKey(privateKey);  

            var block = {
                pubkey: pubkey,  // Chave pública do usuário
                created_at: Math.floor(Date.now() / 1000), // Timestamp atual
                mode: "post", // Tipo de evento (1 = Post)
                query: [
                    ["username", "lanpião.dev"],
                    ["site", "google.com"]
                ],
                content: 'Rei do Cangaço', // Conteúdo da postagem
                app: 'mariabonita.com.br' // Nome do aplicativo que está publicando
            };

            var BlockSigned = await onpostt.signBlock(block, privateKey);
            console.log('Block Assinado:', BlockSigned);
            // BlockSigned.content = 'oalr'
            onpostt.sendBlock(BlockSigned, function(response) {
                console.log('Resposta do relay:', response);
            });
        }
